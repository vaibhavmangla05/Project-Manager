Project Manager

A simple and efficient Project Management System designed to help users organize tasks, track progress, and manage projects seamlessly.

Overview

This project is a project management tool that allows users to:

Create and manage projects
Add, update, and delete tasks
Track project progress
Organize workflows efficiently
Features
Create and manage multiple projects
Task creation and assignment
Progress tracking (To-Do / In Progress / Done)
Edit and delete tasks/projects
Search and filter functionality (optional — update if implemented)
User-friendly interface
